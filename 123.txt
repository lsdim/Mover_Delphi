Дана програма призначена для роботи з файлами:
    1. Копіювання
    2. Переміщення
    3. Видалення
    4. Запуск вказаного додатку при знаходженні певного файлу
    5. Запуск у фоні вказаного додатку при знаходженні певного файлу
    6. Виведення повідомлення при знаходженні вказаного файлу
    7. Перейменування

В полі "Маска" можна використовувати спеціальні символи:
    ? - будь-який символ,
    * - будь-яка кількість будь-яких символів.
Також можна використовувати спеціальні теги для маски:
    <d> - поточне число,
    <d-1> - вчорашнє число,
    <d-2> - позавчорашнє число,
    <m> - поточний місяць,
    <m-1> - минулий місяць,
    <m-2> - позаминулий місяць,
    <yy> - поточний рік (останні дві цифри, наприклад 2013 -> 13),
    <yyyy> - поточний рік (чотири цифри).

Приклад: 45008<d-1><m><yy>.dbf

Також в полі "МАСКА" можна використовувати теги щоб вказати 
день або дні виконання вибраного завдання
    <пн> - вибране завдання буде виконуватись в понеділок,
    <вт> - вибране завдання буде виконуватись у вівторок,
    <ср> - вибране завдання буде виконуватись в середу,
    <чт> - вибране завдання буде виконуватись в четвер,
    <пт> - вибране завдання буде виконуватись в п'ятницю,
    <сб> - вибране завдання буде виконуватись в суботу,
    <нд> - вибране завдання буде виконуватись в неділю,

Також можна вказати час або проміжок часу у форматі <HH:mm>
наприклад:
    <12:05>              - вибране завдання буде виконуватись о 12:05 
один раз,
    <12:05><14:30> - вибране завдання буде виконуватись з 12:05 по 14:30 через вказаний інтервал часу,

Всі теги заключаються між знаками "<",">" (напочатку і вкінці відповідно)

Теги для маски можна використовувати і полях "МІСЦЕ ПРИЗНАЧЕННЯ" та "ПЕРЕВІРЯТИ ПАПКИ"

При виборі операції "ЗАПУСТИТИ ФАЙЛ" у полі "МІСЦЕ ПРИЗНАЧЕННЯ" потрібно вказати додаток, який буде виконуватися при знаходженні файлу, який відповідає заданій масці.

При виборі операції "ПЕРЕЙМЕНУВАТИ" у полі "МІСЦЕ ПРИЗНАЧЕННЯ" потрібно вказати шаблон:

[стара назва]%[нова назва]

Наприклад для файлу 4500120032013.dbf:
  45001%45008 - в назві файлу буде знайдено частину "45001" і замінено її на "45008"
В результаті буде: 4500820032013.dbf.
